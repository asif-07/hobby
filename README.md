# GST Calculator

This project provides a small web-based calculator for computing Goods and Services Tax (GST). It supports amounts that are exclusive of GST (GST added on top) or inclusive of GST (GST already included).

## Usage

1. Open `index.html` in your web browser.
2. Enter the amount and GST rate.
3. Choose whether the amount is **exclusive** or **inclusive** of GST.
4. Click **Calculate** to see the GST value and the total or base amount.

No additional setup or dependencies are required; the app runs entirely in the browser.
